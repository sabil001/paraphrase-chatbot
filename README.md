#Paraphrase Chatbot

Chatbot AI untuk parafrase teks agar menghindari plagiarisme.

## Tech Stack

- Node.js
- Express.js
- Gemini API
- HTML CSS Javascript

## Features

- Paraphrase text
- Highlight perubahan kata
- Chat UI modern
- Upload file

## Installation

```bash
npm install
node index.js

Server berjalan di:

http://localhost:3000
